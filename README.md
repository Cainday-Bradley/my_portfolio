## 🖥️ Getting Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/Cainday-Bradley/my_portfolio.git
   cd my_portfolio
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   ```
4. **Open in your browser:**
   Visit [http://localhost:5173](http://localhost:5173)
